Portfolio Website for Ashwatha E Sureshkumaar

Description

This is a personal portfolio website for showcasing the professional journey, projects, skills, and achievements of Ashwatha E Sureshkumaar. 
It is designed to provide an overview of education, work experience, testimonials, and certificates. The website is responsive and visually 
appealing for various screen sizes, including desktops, tablets, and smartphones.

Features

Responsive Design: Optimized for all devices, including iPads and smartphones.

Interactive Navigation: Smooth scroll and active section highlights.

Projects Section: Showcases major projects with descriptions and GitHub links.

Certificates Section: Highlights professional certifications with external links.

Testimonials: Displays feedback from colleagues and mentors.

Contact Section: Includes a contact form and quick links for direct communication.

Tags Used

Here are the HTML tags used in the website, along with their purpose:

Structural Tags

<html>: Defines the root of the HTML document.

<head>: Contains metadata, links, and scripts.

<body>: Encloses all the visible content of the website.

<header>: Defines introductory content or navigation links.

<footer>: Represents the footer section of the website.

<nav>: Contains navigation links.

<section>: Groups content into thematic sections.

<article>: Used for self-contained content like the certificates table.

Content Tags

<h1>, <h2>, <h3>: Define headings for different sections.

<p>: Paragraphs for textual content.

<ul>, <li>: Unordered lists for listing items like skills or quick links.

<table>, <thead>, <tbody>, <tr>, <th>, <td>: Used for tabular data in the certificates section.

<a>: Hyperlinks to external pages or sections within the site.

<img>: Embeds images, used in the gallery and social media icons.

Form Tags

<form>: Defines the contact form.

<input>: Input fields for text, email, etc.

<textarea>: For multi-line text input.

<button>: Submit button for the form.

Media Tags

<embed>: Embeds the resume PDF.

Meta Tags

<meta>: Provides metadata about the HTML document (e.g., charset, viewport, SEO).

Script Tags

<script>: Used to link external JavaScript files for interactivity.

Technologies Used

HTML5: For structure and semantic content.

CSS3: For styling and responsive design.

JavaScript: For interactivity and animations.
